Integrated results

Kinase co-regulatory networks constructed from kinase-kinase interactions between top-ranked kinase results. Directed edges indicate interactions supported by kinase-substrate evidence; undirected edges indicate protein-protein interaction or unspecified interaction evidence only. Hover over an edge to view libraries in which evidence for an interaction was found. Networks can be downloaded in SVG or PNG format using the links below each graph.

Mean rank

Interactions between top-ranked kinases generated using the MeanRank method.

Top rank

Interactions between top-ranked kinases generated using the TopRank method.

Kinase-substrate interaction

Kinase co-regulatory networks constructed from top-ranked kinase enrichment results for various kinase-substrate interaction libraries. Directed edges indicate interactions supported by kinase-substrate evidence; undirected edges indicate protein-protein interaction or unspecified interaction evidence only. Hover over an edge to view libraries in which evidence for an interaction was found. Networks can be downloaded in SVG or PNG format using the links below each graph.

Cheng KSIN

Kinase-substrate interaction network constructed by Cheng et al. from Phospho.ELM, HPRD, PhosphoNetworks, and PhosphoSitePlus.

PTMsigDB

Phosphosite signatures of kinase activities, perturbations, and signaling pathways curated from literature.

PhosD All

Kinase libraries generated by training the PhosD model on Phospho.ELM, PhosphoSitePlus, and multiple datasets.

Protein-protein interaction

Kinase co-regulatory networks constructed from top-ranked kinase enrichment results for various protein-protein interaction libraries. Directed edges indicate interactions supported by kinase-substrate evidence; undirected edges indicate protein-protein interaction or unspecified interaction evidence only. Hover over an edge to view libraries in which evidence for an interaction was found. Networks can be downloaded in SVG or PNG format using the links below each graph.

prePPI

High-confidence protein-protein interaction predictions generated from a Bayesian network relying on 3D structural information as well as functional, evolutionary, and expression information.

BioGRID

Multi-validated protein-protein interactions from high- and low-throughput experiments manually curated from literature.

mentha

Molecular interactions aggregated from MINT, IntAct, BioGRID, MatrixDB, and DIP.

MINT

Protein-protein interactions from high- and low-throughput experiments manually curated from literature.

HIPPIE

Aggregated experimentally-determined protein-protein interactions from IntAct, MINT, BioGRID, HPRD, DIP, BIND, and MMPI.

STRING bind

Subset of the STRING database which includes only interactions annotated as involving physical binding.

Cheng PPI

Protein-protein interaction network constructed by Cheng et al. from IntAct, MINT, BioGRID, DIP, HPRD, and MIPS MPact.
