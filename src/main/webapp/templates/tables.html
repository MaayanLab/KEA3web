Integrated results

Integrated rankings of kinases across libraries based on two different metrics. Clicking on a name in the “Protein” column will open a Harmonizome page for the relevant protein-coding gene. Select a column heading to order the results by the values in that column; select an already active column heading to toggle between ascending and descending order.

Mean rank

Integrated kinase ranking based on the MeanRank metric, which gives the average rank of each kinase across all libraries containing that kinase. Click a value in the “Mean rank” column to see individual library rankings for a kinase.

Top rank

Integrated kinase ranking based on the TopRank metric, which gives the best-scaled rank of each kinase across all libraries containing that kinase. Click a value in the “Integrated scaled rank” column to display the library in which the kinase was ranked highest.

Kinase-substrate interaction

Kinase enrichment analysis results for different kinase-substrate interaction libraries. By default, kinases are ranked from lowest Fisher’s exact test (FET) p-values to highest, with higher ranking indicating more significant substrate overlap between the query set and gene sets in the library. False discovery rates (FDRs) were computed using the Benjamini-Hochberg correction.

Cheng KSIN

Kinase-substrate interaction network constructed by Cheng et al. from Phospho.ELM, HPRD, PhosphoNetworks, and PhosphoSitePlus.

PTMsigDB

Phosphosite signatures of kinase activities, perturbations, and signaling pathways curated from literature.

PhosD All

Kinase libraries generated by training the PhosD model on Phospho.ELM, PhosphoSitePlus, and multiple datasets.

Protein-protein interaction

Kinase enrichment analysis results for different protein-protein interaction libraries. By default, kinases are ranked from lowest Fisher’s exact test (FET) p-values to highest, with higher ranking indicating more significant substrate overlap between the query set and gene sets in the library. False discovery rates (FDRs) were computed using the Benjamini-Hochberg correction.

prePPI

High-confidence protein-protein interaction predictions generated from a Bayesian network relying on 3D structural information as well as functional, evolutionary, and expression information.

BioGRID

Multi-validated protein-protein interactions from high- and low-throughput experiments manually curated from literature.

mentha

Molecular interactions aggregated from MINT, IntAct, BioGRID, MatrixDB, and DIP.

MINT

Protein-protein interactions from high- and low-throughput experiments manually curated from literature.

HIPPIE

Aggregated experimentally-determined protein-protein interactions from IntAct, MINT, BioGRID, HPRD, DIP, BIND, and MMPI.

STRING bind

Subset of the STRING database which includes only interactions annotated as involving physical binding.

Cheng PPI

Protein-protein interaction network constructed by Cheng et al. from IntAct, MINT, BioGRID, DIP, HPRD, and MIPS MPact.

All associations

STRING

Database of direct and indirect protein-protein interactions, including physical interaction, co-expression, co-occurrence in literature, evolutionary co-occurrence, and other association types
